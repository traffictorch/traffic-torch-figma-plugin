# Traffic Torch AI GEO & SEO Tools for Figma

**Instant 360° SEO, UX, GEO & AI health analysis directly inside Figma.**

Educational plugin that brings the full power of [traffictorch.net](https://traffictorch.net) into your design workflow — no more switching tabs.

### Features
- Clean tabbed interface (Welcome, All Tools, Help Guides)
- Full Traffic Torch tool list with educational descriptions
- One-click launch with your live site URL pre-filled (`?url=...`)
- Always-visible green Launch button with loading state
- Dark/light mode support (gray-800 / dark-gray-200 text + #10b981 accent)
- Mobile-friendly layout inside the 420px modal
- Built with latest 2026 Figma Plugin API + plain HTML for maximum reliability

### How to Install
1. Clone or download this repo
2. Run `npm install && npm run build`
3. In Figma → **Plugins** → **Development** → **Import plugin from manifest** → select `manifest.json`
4. Run "Traffic Torch AI GEO & SEO Tools for Figma"

### How to Use
1. Open any Figma file
2. Run the plugin
3. Select a tool from the dropdown
4. Type or paste your live website URL (e.g. `yoursite.com`)
5. Click **🚀 Launch Tool with URL →**
6. Traffic Torch opens with your URL pre-filled and full educational report

**Tip:** The plugin works best with live production websites. For best results, use your final deployed site URL.
